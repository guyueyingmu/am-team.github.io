Use 开始使用

include 引入方式

  • 2种引入方式

    • 线上源文件
    • SPM3模块

      cdn-source 线上源文件

  • 1. 引入AMUI9.5推荐库

CDN源文件地址：https://as.alipayobjects.com/g/h5-lib/antui/9.5.3/amui.css
离线资源包地址：https://a.alipayobjects.com/amui/native/9.0/amui.css [离线包中样式文件暂未更新]

相关代码：

<link href="https://as.alipayobjects.com/g/h5-lib/antui/9.5.3/amui.css" rel="stylesheet"/>

AMUI9.5库已集成组件查看，传送门

  • 2. 引入单独某个模块

以button为例：
https://as.alipayobjects.com/g/h5-lib/antui/9.5.3/widget/button.css

橙色部分，可以按照对应组件路径修改

某些模块有一定依赖，如widget/message，依赖icon/message

https://as.alipayobjects.com/g/h5-lib/antui/9.5.3/icon/message.css
https://as.alipayobjects.com/g/h5-lib/antui/9.5.3/widget/message.css
  • 3. 多模块合并

使用CDN的拼接合并功能：

https://as.alipayobjects.com/g/h5-lib/antui/9.5.3/??amui.css,icon/message.css,widget/message.css

spm3-module SPM3模块

  • 1. 先安装本模块

需要安装spm3

spm install ant-ui --save
  • 2. 引用完整样式库
@import 'ant-ui';
  • 3. 引入单独某个模块
@import 'ant-ui/widget/button.css';
  • 4. 多模块合并
@import 'ant-ui/amui.css';
@import 'ant-ui/icon/message.css';
@import 'ant-ui/widget/message.css';
