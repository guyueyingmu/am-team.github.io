Button 按钮

preview 效果预览

usage 使用说明

  • AntUI默认已集成
  • 模块名：widget/button
  • 可选依赖：util/flexbox（布局，amui包已集成）

使用方式

default 基础样式

  • 基础特征
    • 颜色蓝色(blue)
    • 尺寸高度(42px)，宽度100%
    • 标签建议button或者a
蓝色按钮 
<button type="button" class="am-button">蓝色按钮</button>
<a href="#" class="am-button">蓝色按钮</a>

color 颜色扩展

  • 可选颜色(4种)：
    • 蓝色(mode值：blue)
    • 红色(mode值：red)
    • 白色(mode值：white)
    • 黑色(mode值：black)
<button type="button" class="am-button" am-mode="blue">蓝色按钮</button>
<button type="button" class="am-button" am-mode="red">红色按钮</button>
<button type="button" class="am-button" am-mode="white">白色按钮</button>
<button type="button" class="am-button" am-mode="black">黑色按钮</button>

size 尺寸扩展

  • 可选尺寸(5种)：
    • 26px(mode值：tiny)
    • 31px(mode值：little)
    • 36px(mode值：small)
    • 40px(mode值：middle)
    • 42px(default)
<button type="button" class="am-button" am-mode="red tiny">红色26px按钮</button>
<button type="button" class="am-button" am-mode="white little">白色31px按钮</button>
<button type="button" class="am-button" am-mode="black small">黑色36px按钮</button>
<button type="button" class="am-button" am-mode="blue middle">蓝色40px按钮</button>
<button type="button" class="am-button" am-mode="blue">默认42px按钮</button>

disabled 按钮可用性

  • 可选类型(3种)：
    • disabled(通用不可用，最常用)
    • blue-disabled(不推荐，少数场景使用)
    • red-disabled(不推荐，少数场景使用)
<button type="button" disabled="disabled" class="am-button" am-mode="red">正常按钮 不可用</button>
<button type="button" class="am-button" am-mode="red disabled">看起来 不可用</button>
<button type="button" disabled="disabled" class="am-button" am-mode="blue blue-disabled">特殊不可用蓝色按钮</button>
<button type="button" disabled="disabled" class="am-button" am-mode="red red-disabled">特殊不可用红色按钮 </button>

light 简易按钮风格

<button type="button" class="am-button" am-mode="blue light small">蓝色简易按钮</button>
<button type="button" class="am-button" am-mode="white light small">白色简易按钮</button>
<button type="button" class="am-button" am-mode="light small disabled">不可用简易按钮</button>

layout 布局

  • 可选依赖：util/flexbox（布局，amui包已集成）
行内按钮 最小号按钮 
<div class="am-flexbox" am-mode="average">
    <div class="am-flexbox-item">
        <button type="button" class="am-button" am-mode="blue">均长按钮</button>
    </div>
    <div class="am-flexbox-item">
        <button type="button" class="am-button" am-mode="white">长短按钮均长</button>
    </div>
</div>
<a href="button" class="am-button" am-mode="white inline">行内按钮</a>
<a href="button" class="am-button" am-mode="tiny light inline">最小号按钮</a>
