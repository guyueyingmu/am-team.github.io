        ap.showTabBar(OPTIONS, CALLBACK)

        showTabBar控件，固定在界面底部

        OPTIONS 参数说明

        名称类型必填描述
        colorString否tab文字颜色，默认 #00000
        activeColorString否tab激活文字颜色，默认 #00000
        activeIndexNumber否默认激活的 tab 索引，默认 0
        itemsObject Array是tab 配置数组，数组中每个项是一个对象。tab 的具体配置字段见下表

        items 配置项参数说明：

        名称类型必填描述
        titleString是tab 标题
        iconString是tab 图标，支持 base64
        activeIconString是tab 激活图标，支持 base64
        badgeString否tab 红色气泡，默认 -1。其中 0 表示小红点，-1 表示不显示，其他值展示出来

        CALLBACK 参数说明

        点击 tab 时调用

        名称类型描述
        indexNumber被点击的 tab 的索引，从0开始
